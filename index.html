Features

Everything you need,
nothing you don't.

Built for Arch users who care about both power and aesthetics.

📦

pacman + AUR, unified

Search both pacman repos and AUR simultaneously. Results are deduplicated, with pacman taking priority. One-click install and uninstall with real-time progress events.

⭐

Trustpilot-style reviews

Rate apps 1–5 stars, leave comments, like useful reviews. Anti-fake protection: JWT auth, one review per user per package, rate limiting, and no self-voting.

🔐

Full auth system

Local accounts with Argon2id hashing. Discord and GitHub OAuth. JWT + rotating refresh tokens. Account lockout after 5 failed attempts. Password reset via email OTP.

📸

Snapshot before install

Automatically creates a Timeshift or Snapper snapshot before every install and uninstall. Broke your system? Roll back in seconds.

🏷️

Source badges

Official pacman packages get a green ✓ Verified badge. AUR packages are clearly marked. Low-rated apps (below 3.5★) are filtered from homepage recommendations.

🌏

Multilingual

Switch between English, Tiếng Việt, and 日本語 without restarting the app. Preference is saved in localStorage. Adding a new language takes one JSON file.

Security

Built secure by default.

12 layers of protection — from the desktop app to the Cloudflare edge.

🔑

X-Api-Key header

Every request carries a shared secret. Requests without it return 401. Timing-safe HMAC comparison — no timing attacks.

🧂

Argon2id passwords

t=2, m=512KB, p=1 — adapted for Cloudflare Workers 10ms CPU limit. Constant-time verification.

🔄

Refresh token rotation

Only SHA-256 hash stored in D1. Raw token never persists server-side. Revoked on password reset.

🚫

Shell injection prevention

Package names validated against [a-zA-Z0-9\-_.+@] whitelist on both Go and Worker side.

⏱️

Rate limiting

5 reviews per user per 24h. D1-backed, per-endpoint. Returns 429 with Retry-After header.

🔒

pkexec GUI auth

System changes use pkexec for a GUI authentication dialog. No terminal password exposure.

Get Started

Up and running in 60 seconds.

Two paths: grab the AppImage and go, or deploy your own backend.

★ RECOMMENDED

Run the AppImage

Single executable, no installation required. Set your Worker URL and API key, then run.

# Download from GitHub Releases
chmod +x ArchStore-*.AppImage

export ARCHSTORE_CF_WORKER_URL="https://..."
export ARCHSTORE_API_KEY="your-key"

./ArchStore-*.AppImage
SELF-HOST

Deploy your own backend

Fork the repo, deploy your own Cloudflare Worker, and build the app pointing to it.

# 1. Deploy Cloudflare Worker
cd cloudflare/worker
cp wrangler.toml.example wrangler.toml
wrangler d1 create archstore-db
wrangler deploy

# 2. Build the desktop app
wails dev
